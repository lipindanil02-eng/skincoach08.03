# SkinCoach — Полный роадмап продукта

## Текущее состояние
- MVP работает на Railway
- 8-слойный пайплайн (quality → vision → reasoning → questions → triage → recommendations → safety → response)
- EfficientNet-B3 модель (HAM10000, ~20k фото)
- LLM через OpenRouter (Nemotron, Arcee, StepFun)
- history.json хранит состояния пользователей

## Главная ценность продукта

> понимают его кожу, и каждый следующий этап логично продолжает предыдущий.
